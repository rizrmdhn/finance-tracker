# Finance Tracker — Android Releases

This repository hosts Android APK releases for **Finance Tracker**.
Source code and desktop releases live in [rizrmdhn/finance-tracker](https://github.com/rizrmdhn/finance-tracker).

## Download

Go to the [Releases](../../releases) page and download the latest APK for your device.

> **Note:** Finance Tracker is not distributed via the Play Store.
> You need to enable **Install from unknown sources** on your device before installing.

### Enable unknown sources

- **Android 8+**: Settings → Apps → Special app access → Install unknown apps → (your browser or file manager) → Allow

## Installation

1. Download the `.apk` file from the latest release
2. Open the downloaded file on your Android device
3. Tap **Install** and follow the prompts
4. Open **Finance Tracker** from your app drawer

## Updates

The app checks for new releases automatically from the Settings page.
You can also watch this repository (Watch → Custom → Releases) to get notified when a new version is available.

## Versioning

Releases follow [Semantic Versioning](https://semver.org): `MAJOR.MINOR.PATCH`

| Tag | Meaning |
|-----|---------|
| `mobile/v1.0.0` | Stable release |
| `mobile/v1.1.0-beta` | Beta / pre-release |

## Requirements

- Android 8.0 (API 26) or higher
- ~150 MB free storage
